Components consist of:

  • Atom
  • Block
  • Collection
  • Template
  • Page

Atom

An atom is the smallest possible breakdown of a component.

Block

A block is a reusable component which consists of more than 1 atom.

Collection

A collection is a reusable component made up of a block AND an atom OR another block.

Template

A template is made up of a reusable layout of components.

A layout requires at least one component, where that component can be replaced with another component, and the template layout will maintain integrity.

A template is not concerned about the components it consists of. It concerns itself with their visual structure.

Page

A page is how you get to a layout. It’s the starting point for choosing components.

A page is not a component, nor is a page reusable.

It’s a specific instance of a combination of components.
